# Java Graph Traversal Project  
## Breadth-First Search (BFS) & Depth-First Search (DFS)

---

## Overview

This project provides a structured implementation of two fundamental graph traversal algorithms: **Breadth-First Search (BFS)** and **Depth-First Search (DFS)** using Java.

The application allows manual graph creation by entering vertices and edges from the console. Traversal operations are executed from a user-defined starting node, and the results are displayed clearly in the terminal.

---

## Features

- Manual graph construction (user-defined input)
- Adjacency List representation
- BFS traversal using Queue
- DFS traversal using recursion
- Console-based execution
- Clear traversal output

---

## Algorithm Details

### Breadth-First Search (BFS)
- Explores nodes level by level
- Uses Queue data structure
- Suitable for shortest path in unweighted graphs
- Time Complexity: O(V + E)

### Depth-First Search (DFS)
- Explores deeply before backtracking
- Implemented recursively
- Useful for connectivity and cycle detection
- Time Complexity: O(V + E)

## How to Run

1. Compile:
```
javac Main.java
```

2. Run:
```
java Main
```

3. Enter:
- Number of vertices
- Edges manually
- Starting vertex

The program outputs BFS and DFS traversal results.
